## 2026/02/13
### endian

エンディアンとは、2バイト以上の整数データを
メモリにどの順番で並べるかのこと。

1バイトの最大値は
255 = 11111111 = 0xFF

0x100（256）になると2バイト必要。

例：
0x1234 は 2バイトなので

0x12 0x34 に分けられる。

これをメモリに置いたときに低いアドレスから

0x12 0x34　と並べるとビッグエンディアン

逆に

0x34 0x12　と並べるとリトルエンディアンという。

### 8進数の基数変換

0000012   （8進数)  
自分の答え：1*8^1 + 2*8^0 = 24　☓  
正解：10  
2*8^0 を 16 と勘違いしていた。  
n^0 は 1 になることを忘れていた。  　
→ 1×8 + 2×1 = 10  

0000177   (8進数)  
自分の答え: 1*8^2 + 7*8^1 + 7*8^0 = 64 + 56 + 7 = 127 正解  

0000754   （8進数）  
自分の答え：7*64 + 5*8 + 4 = 448 + 40 + 4 = 492　正解  

8進数 100 = 64  
8進数 200 = 128  
8進数 400 = 256
